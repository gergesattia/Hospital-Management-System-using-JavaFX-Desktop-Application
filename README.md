# Hospital Management System (JavaFX Desktop Application)

Hospital Management System is a JavaFX desktop application designed to manage hospital operations efficiently. It provides an easy-to-use interface for managing patients, doctors, appointments, medical records, billing, and administrative tasks. The system improves data organization, reduces manual work, and ensures secure database management.

## Quick Setup

1. **Build with Maven:**
   ```bash
   mvn clean package
   ```

2. **Run the application:**
   Launch it from your IDE or execute the generated JAR file.

## Developer Notes

- Do NOT commit sensitive files in the `tokens/` folder. They are already ignored by `.gitignore`.
